# 🎵 Spotify Clone

A responsive Spotify Clone built using **HTML5** and **CSS3**, inspired by Spotify's desktop web player. This project recreates the user interface with a modern layout using Flexbox and responsive design principles.

## ✨ Features

- 🎵 Spotify-inspired user interface
- 📂 Responsive sidebar navigation
- 🎧 Music player with playback controls
- 💿 Album information section
- 📱 Responsive layout using Flexbox
- 🎨 Clean and modern UI design
- ⭐ Hover effects and interactive icons

1. Clone the repository


2. Open the project folder

3. Open `index.html` in your browser.
